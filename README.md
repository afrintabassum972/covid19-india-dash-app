# 🩺 Coronavirus Pandemic Analysis Dashboard

A data-driven interactive web application built with **Dash** and **Plotly** to monitor the COVID-19 landscape in India. This project translates raw data into actionable insights through dynamic visualizations.

## 📊 Project Preview
![Dashboard Screenshot](screenshots/Screenshot%202026-03-09%20080214.png)
> Interface featuring real-time cards, commodity trends, case distribution, and state-wise analysis.

## 🌟 Key Objectives
* **Live Metrics:** Instant visibility into Total, Active, Recovered, and Deceased cases.
* **Deep-Dive Analysis:** State-level filtering to understand regional impact.
* **Zone Tracking:** Visual classification of Red, Green, Orange, and Blue zones.
* **Resource Management:** Monitoring essential supplies like Masks, Sanitizers, and Oxygen.

## 🛠️ Tech Stack
* **Python:** Core logic and backend.
* **Pandas:** High-performance data cleaning and manipulation.
* **Plotly & Dash:** For creating the interactive charts and web layout.
* **Bootstrap:** Ensuring the dashboard is responsive and mobile-friendly.

## 📂 Dataset Insight
* **File:** `state_wise_daily data file IHHPET.csv`
* **Fields:** Confirmed, Recovered, Deceased, Hospitalised, Medical Supplies (Masks/Oxygen), and Zone Data.

## 🚀 Main Features
* **Real-time KPI Cards:** High-level summary showing the current pandemic status across the nation.
* **Commodities Total Count:** A detailed visualization tracking the total volume of essential medical supplies.
* **Case Distribution:** Interactive pie charts illustrating the ratio of Confirmed, Recovered, and Deceased cases.
* **State-wise Analysis:** Comprehensive bar graphs providing a regional breakdown of COVID-19 cases.

## ⚙️ Installation & Setup
1. **Clone the repo:** 
   `git clone https://github.com`
2. **Install dependencies:** 
   `pip install -r requirements.txt`
3. **Run the app:** 
   `python app.py`
